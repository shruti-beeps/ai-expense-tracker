AI Expense Tracker 💰🤖

An AI-powered Expense Tracker built using Django that helps users manage daily expenses with smart automatic expense categorization.

🚀 Features
User Registration & Login Authentication
Add Expenses
Edit Expenses
Delete Expenses
AI-based Automatic Category Detection
Total Expense Calculation
Clean and Responsive UI
Secure User-wise Expense Management


🛠️ Technologies Used
Python 3.14
Django
HTML
CSS
SQLite
Git & GitHub

⚡ AI Feature

The project automatically detects categories based on expense title.

Example
Expense Title	Category
Pizza	Food
Uber	Travel
Movie	Entertainment
Others	Other


▶️ How to Run the Project
1️⃣ Clone Repository
git clone https://github.com/shruti-beeps/ai-expense-tracker.git
2️⃣ Open Project Folder
cd ai-expense-tracker
3️⃣ Install Django
pip install django
4️⃣ Run Migrations
py -3.14 manage.py migrate
5️⃣ Start Server
py -3.14 manage.py runserver
🌐 Open in Browser
http://127.0.0.1:8000/
